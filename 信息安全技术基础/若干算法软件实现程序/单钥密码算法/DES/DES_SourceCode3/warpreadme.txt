Warp.exe Readme

NOTICE: 

Any changes made to an encrypted file will make it corrupted.

Please preserve any previous versions of file "warpdll.dll", unless you didn't encrypt any data with any of them. The difference in the version number before the dot shows that these two versions aren't compatible.

Your key can up to 27 bytes, including any visual charactors. If your key contains Chinese charactors, each Chinese charactor takes up two bytes.

When your key isn't protected by "*", you can enter Chinese charactors.

The max file size this software can handle is 4G.


注意：

对加密过的文件的任何改动将会使该文件毁坏，不能完整解密。

请保留以前版本的“warpdll.dll”，除非你没有用以前版本的“warpdll.dll”加密任何数据。版本号的小数点前面的数字的不同说明这两个版本不兼容。

密钥最多可以有27个字符，可以是任何可打印字符。每个中文占用两个字符。

当密钥没有被“*”保护时，你可以输入中文密钥。

本软件所能处理的最大文件是4G。
