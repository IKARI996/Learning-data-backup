AES Calculator

You can use the AES Calculator applet displayed below to encrypt or decrypt using AES the specified 128-bit (32 hex digit) data value with the 128/192/256-bit (32/48/64 hex digit) key, with a trace of the calculations. Some example values which may be used are given below.

Example AES test values (taken from FIPS-197) are:

Plaintext: 00112233445566778899aabbccddeeff
Ciphertext: 69c4e0d86a7b0430d8cdb78070b4c55a

Plaintext: 00112233445566778899aabbccddeeff

Encrypting the plaintext with the key should give the ciphertext, decrypting the ciphertext with the key should give the plaintext.
