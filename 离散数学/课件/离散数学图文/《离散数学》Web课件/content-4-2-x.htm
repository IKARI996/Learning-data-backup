本 节 习 题

1.设集合 A={A1,A2,...,An} 是集合 S 的划分，并且 B 是一个任意集合。试证明集合{A1B,A2B,...AnB}是集合 SB 的划分。

2.把 n 个元素的集合划分成两个类，共有多少种不同的方法？

3.有一个集合公式，它包含有两个集合变元 X 和 Y,还有集合运算、 和 ~。证明：能够求出另外一个集合公式，它等于给定集合公式，且仅由极小项的联合组成。

4.证明：对于集合公式来说，运算集合{，~}是全功能的。

5.给定一个集合 X，并且 R 是 X 中的一种关系。对于所有的 ，，X 来说，如果 R 和 R 蕴含 R,则称 R 是个循环关系。证明：R 是一种等价关系，当且仅当关系 R 是自反的和循环的。

6.设 R1 和 R2 都是集合 X 中的等价关系。证明：划分 C1 中的每一个等价类都包含于划分 C2 的某一个等价类之中，当且仅当有 R1R2。

7.设 R1 和 R2 都是集合 X 中的等价关系，并且 R1 和 R2 分别具有秩 r1 和 r2。证明 R1R2 也是集合 X 中的一种等价关系，它的秩至多为 r1*r2。再证明，R1R2不一定是集合 X 中的一种等价关系。
