由等价关系来进行划分

等价类
设 R 是集合 X 的等价关系，对于任何 xX 来说，定义集合X 为：
={y|(yX) (yRx)}
则称集合是由 xX 生成的关系 R 的等价类。

 

商集
设 X 是一个非空集合，R 是 X 中的等价关系。称 R 等价类的集合 {|xX} 是个 X 对 R 的商集。并记为 X/R，也可以写成 X(mod R), 亦即：
X/R={|xX}
这里 X/R 是一个集合，它的基数是 X 的各元素所生成的不同的 R 等价类的数目，也称为是等价关系 R 的秩。

 

由等价关系来进行划分
定理：设 X 是一个非空集合，R 是 X 中的等价关系。则集合 X 对 R 的 商集 X/R 是 X 得一种划分。
