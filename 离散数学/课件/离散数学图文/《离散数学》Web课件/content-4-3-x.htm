本 节 习 题

1.给定集合 X={0,1,2,3}，且 X 中有关系：
R1={<i,j>|(i,jX)((j=i+1)(j=i/2))}
R2={<i,j>|(i,jX)(i=j+2)}
求下列合成关系：
(1)R1R2
(2)R2R1
(3)R1R2R1
(4)R1R1R1
(5)R2R2R2

2.给定集合 X={a,b,c,d}，且 X 中有二元关系：
R1={<a,a>,<a,b>,<b,d>}
R2={<a,b>,<b,c>,<b,d>,<c,d>}
求出合成关系 R1R2，R2R1，R1R1 和 R2R2。

3.设 R1，R2 和 R3 是集合 X 中的二元关系。证明如果有 R1R2，那么：
(1)R1R3R2R3
(2)R3R1R3R2

4.设 R1 和 R2 是集合 X 中的任意二元关系。证明或反驳下列命题：
(1)如果 R1 和 R2 是自反的，则 R1R2 也是自反的。
(2)如果 R1 和 R2 是反自反的，则 R1R2 也是反自反的。
(3)如果 R1 和 R2 是对称的，则 R1R2 也是对称的。
(4)如果 R1 和 R2 是反对称的，则 R1R2 也是反对称的。
(5)如果 R1 和 R2 是可传递的，则 R1R2 也是可传递的。

6.给定集合 X，且 R 是 X 中的二元关系。证明 RRR，当且仅当关系 R 是可传递的。

7.设集合 X={0，1，2，3}，并且 X 中有关系
R1={<i,j>|(j=i+1)(j=i/2)}
R2={<i,j>|i=j+2}
构成如下关系的关系矩阵：
(1)R1
(2)R2
(3)R1R2
(4)R2R1
(5)R1R2R1
(6)R1R1R1

8.给定关系 R={<i,j>|(i,jI)(j-i=1)}。分别写出关系 R 和 的关系矩阵。

9.设集合 X={1,2,3}。求出 X 中这样的等价关系 R1 和 R2，使得合成关系 R1R2 也是个等价关系。

10.证明：
(1)如果关系 R 是自反的，则 R 的逆关系也是自反的。
(1)如果关系 R 是反自反的，则 R 的逆关系也是反自反的。
(1)如果关系 R 是对称的，则 R 的逆关系也是对称的。
(1)如果关系 R 是反对称的，则 R 的逆关系也是反对称的。
(1)如果关系 R 是可传递的，则 R 的逆关系也是可传递的。

11.如果 S 是个反对称的关系，则在 S 的关系矩阵中，有多少个非零的元素？

12.假定 是集合 X 中的恒等关系，且 S 是 X 中的任何关系，证明 S 是个相容关系。
