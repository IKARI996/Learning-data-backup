程设考试分为笔试和机试。

笔试包括选择题和手写代码题。
选择题大家可以自行百度，会有那种给一段代码让你写出结果这类的题目，
还有基本运算，比如自加自减，i=1， i+++++i=？

手写代码……一定仔细，括号什么的，声明什么的，
去年考了貌似结构体、排序、文件处理之类的东西，大家可以思考一下……

从文件中读出来，然后排序，再写进去……


机试祝大家好运………………听说你们的TA猴赛雷……

小技巧就是，如果这道题不会做也别空着不写代码，
哪怕你只写个return 0;让程序可以正常运行，都可以拿一部分分数的，
当然如果你控制条件，输出题目给出的固定测试的结果就可以拿更多的分数了……
实在是多的一点分数是一点……
